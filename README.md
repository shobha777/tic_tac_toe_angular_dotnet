# tic_tac_toe_angular_dotnet
# Tic Tac Toe Application

## Overview

This project is a Tic Tac Toe game built using Angular and ASP.NET Core Web API.

## Technology Stack

### Frontend
- Angular
- TypeScript
- HTML/CSS

### Backend
- ASP.NET Core Web API
- C#
- Swagger/OpenAPI

## Implemented Features

- Create Game API
- Get Game API
- Move API foundation
- Repository Pattern
- Service Layer
- Dependency Injection
- Swagger Integration

## Project Structure

backend/
    TicTacToe.Api/

frontend/
    tic-tac-toe-ui/
